项目简介
一个简单的前后端分离的笔记系统，实现了注册，登录，笔记添加，修改，删除等功能。
具体项目描述，请跳转至系统介绍....link..
云部署相关，因为是第一次接触云所以最初是部署在GCP上，更方便练手和学习
后期经过云相关的学习后，项目在AWS上构筑，更接近企业部署

設計方針：
①料金：できるだけ料金が発生しないことを優先し、重要度の高いサービスは利用する。
②目標：できるだけ無料枠の範囲内で、できるだけ多くのサービスを利用します。
③AWSの設計原則を基本に従うこと:
1.安全性 
2.可用性 
3.高性能 
4.コスト最適化 
5.運用の優秀性 
6.持続可能性

系统部分介绍：
系统功能
| 機能名         | 説明                                      |
| -----------    | ----------------------------------------  |
| ユーザー登録   | メールアドレスとパスワードでアカウント作成  |
| ログイン       | 用户登录                |
| ノート一覧表示 | 自分の作成したノートを一覧表示              |
| ノート作成     | タイトルと内容を入力して新規ノートを作成    |
| ノート編集     | 既存のノートを編集可能                     |
| ノート削除     | 不要なノートを削除（確認ダイアログあり）    |

系统技术栈
| 層（コンポーネント） | 技術              |               説明            |   版本
| -------------- | ---------------------- | --------------------------    |
| フロントエンド  | React, Tailwind CSS    | ノートのUI画面、APIとの通信    | Node.js 18.x React 18.2.0 Tailwind CSS 3.4.1
| バックエンド   | Spring Boot (Java)     | REST API、認証、ノート処理など | Spring Boot 3.2.5 Java 21
| データベース   | PostgreSQL             | ユーザー・ノート・タグの保存    |PostgreSQL 16.2
| コンテナ化     | Docker, Docker Compose | 環境構築を自動化・統一         |
| CI/CD          | GitHub Actions         | 自動ビルド・テスト・デプロイ   |
